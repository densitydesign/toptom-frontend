# JARVIS User Interface

## Development notes

### Build

Install dependences:
```
$ yarn install
```

Build application:
```
$ yarn build
```

Only if without available server, start the fake server:
```
$ yarn json
```

Run application:
```
$ yarn live
```

### Development

Install dependences:
```
$ yarn install
```

Watch application:
```
$ yarn watch
```

Only if without available server, start the fake server:
```
$ yarn json
```

Run application:
```
$ yarn live
```
